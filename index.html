Let the following JavaScript plugin do the work for you or get into the CSS playground here
P.S. It is a purely CSS effect.

Installation

<script src="jquery.js"></script>
<script src="fillText.js"></script>
<script>
  $(function(){
      $(".fill-text").fillText({
        color: "black",
        text_bg: "#ccc", 
        duration: "2s",
      });
  });
</script>
...

<div data-sample="Fill_Text" class="element"></div>

Options

With default values:

color : "#ccc",
text_bg : "black",
fill_value : 7,
duration : "2s"
